# ROCKTON

Inspirado no Spotify, o Rockton surgiu com o propósito de cadastrar e organizar músicos, bandas e álbuns. A ideia central é ser uma vitrine digital para artistas, permitindo ao público conhecer suas trajetórias, discografias e colaborações, valorizando a história de cada um.

## 📌 Sobre o Projeto

O Rockton centraliza informações musicais de forma organizada e estruturada, facilitando o cadastro e a visualização de:

- 🎤 Músicos / compositores
- 🎸 Bandas
- 💿 Álbuns
- 🤝 Colaborações entre artistas
- 📚 Discografias
